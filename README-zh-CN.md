<h1 align="center">monorepo</h1>

这是一个单体仓库模板

## 使用

`monorepo` 是该 **github** 仓库名称
`z-monorepo` 是该仓库 `packages/z-monorepo` 项目名称

使用该仓库模板时，
1. 请全局替换 `monorepo`    ->  `你的 **github** 仓库名称`
2. 请全局替换 `z-monorepo`  ->  `packages 目录下项目名称`
