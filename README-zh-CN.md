<h1 align="center">monorepo</h1>

这是一个单体仓库模板

## 使用

`your_repo_name` 是该 **github** 仓库名称
`project_name` 是该仓库 `packages/project_name` 项目名称

使用该单体仓库模板时，
1. 请全局替换 `your_repo_name`    ->  `你的 **github** 仓库名称`
2. 请全局替换 `project_name`  ->  `packages 目录下项目名称`
